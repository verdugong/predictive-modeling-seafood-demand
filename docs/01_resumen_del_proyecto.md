# Resumen del Proyecto

Este proyecto desarrolla un sistema de predicción de demanda para productos del mar usando XGBoost. El objetivo es apoyar la planificación de compras, inventario y abastecimiento mediante análisis exploratorio, entrenamiento supervisado y una API de inferencia.

## Alcance

- Análisis exploratorio de datos.
- Preparación del dataset y variables de negocio.
- Entrenamiento y validación de un modelo XGBoost.
- Exportación de artefactos para inferencia.
- API FastAPI para consumo del modelo.

## Entregables principales

- 3 notebooks de Jupyter.
- Documentación en `docs/`.
- `README.md` y estructura de proyecto reproducible.
- Configuración de dependencias con `uv`.
