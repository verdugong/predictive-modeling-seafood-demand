# Metodología

## Flujo de trabajo

1. Carga y limpieza de datos.
2. Análisis exploratorio.
3. Feature engineering.
4. Entrenamiento temporal del modelo.
5. Exportación de artefactos.
6. Predicción e inferencia.

## Buenas prácticas

- Separación clara entre exploración, entrenamiento e inferencia.
- Uso de partición temporal para evitar fuga de información.
- Versionado de dependencias.
- Eliminación de outputs en notebooks para publicación.
- Registro de configuración necesaria para reproducibilidad.
